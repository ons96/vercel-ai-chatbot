# vercel-ai-chatbot

Multi-provider AI chatbot built with Next.js 14, integrating OpenAI, Anthropic, Gemini, tool calls (Tavily/Brave search), and puter.js for cloud execution. Supports model dropdowns, rate limit fallbacks, and extensible configs.

## Setup
- Clone: `git clone https://github.com/ons96/vercel-ai-chatbot.git`
- Install: `npm install`
- Run: `npm run dev`
